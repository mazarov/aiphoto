## Рекомендуемые доработки (без новых URL)

### 3.1 Title / H1 / Description

| **metaDescription** | Готовые промты для фото и ИИ-фотосессии девушки на русском. Женские промты для Nano Banana и нейросетей: портрет, студия, день рождения, с цветами — бесплатно. |

### 3.2 Intro (черновик)

**Абзац 1:**  
Готовые промты для фото девушки и женские промты для ИИ-фотосессии — на русском, бесплатно. Подборка для Nano Banana (нано банана), Gemini, ChatGPT и других нейросетей: реалистичные портреты, студийные кадры и тематические образы.

**Абзац 2 (popularLinks или второй абзац):**  
Популярные сценарии: [день рождения](/promty-dlya-foto-devushki/den-rozhdeniya/), [с цветами](/promty-dlya-foto-devushki/s-cvetami/), [портрет](/promty-dlya-foto-devushki/portret/) и [студийное фото](/promty-dlya-foto-devushki/studiynoe/), [с машиной](/promty-dlya-foto-devushki/s-mashinoy/), [чёрно-белое](/promty-dlya-foto-devushki/cherno-beloe/). Для парных кадров — [промты для пар](/promty-dlya-foto-par/).

**Реализация:** plain `intro` + optional `popularLinks[]` в `SeoContent` (как у `s_mashinoy` — без JSX в intro).

### 3.3 FAQ (расширить до 7–8)

1. **Есть готовые промты для ИИ-фотосессии девушки на русском?** — да, все тексты на русском; копируй и вставляй в Nano Banana или другую нейросеть.
2. **Какие промты подходят для Nano Banana, Gemini и ChatGPT?** — промты универсальны; для Nano Banana лучше загружать своё фото.
3. **Где промты для фото девушки на день рождения?** — подборка [День рождения](/promty-dlya-foto-devushki/den-rozhdeniya/): торт, свечи, возраст.
4. **Как сделать реалистичное или красивое фото девушки в нейросети?** — промты с «фотореалистичный», «студийный свет», «высокая детализация».
5. **Есть промты для фото девушки с машиной, с цветами или в студии?** — да: [с машиной](/promty-dlya-foto-devushki/s-mashinoy/), [с цветами](/promty-dlya-foto-devushki/s-cvetami/), [студийное](/promty-dlya-foto-devushki/studiynoe/).
6. **Можно ли использовать промты для обработки своего фото?** — да, см. [промты для обработки фото](/promty-dlya-obrabotki-foto/).
7. **Промты бесплатные?** — сохранить текущий ответ.
8. **Нужны промты для парного фото?** — [промты для пар](/promty-dlya-foto-par/), не смешивать с одиночными.
