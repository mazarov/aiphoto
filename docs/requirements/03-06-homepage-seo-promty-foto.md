## 1. Продукт и цели

### 1.1 Задача

Заточить **главную страницу** promptshot.ru под семантический кластер «промты для фото» — каталог готовых текстовых промтов для генерации фото в ИИ-нейросетях.

Страница уже существует как каталог карточек; задача — **усилить SEO-copy и schema**, не меняя продуктовую логику каталога.

### 1.2 Цели (acceptance на уровне продукта)

| ID | Цель | Критерий успеха |
|----|------|-----------------|
| **G1** | Head + СЧ keyword | Title — «промты для фото ИИ»; H1 — «Промты для фото и ИИ-фотосессии в нейросетях» |
| **G2** | СЧ-кластеры | Description, hero, intro и FAQ закрывают «промты для ии фотосессии», «промт для создания фото», «промты для фото ии» |
| **G3** | Чистота intent | Убраны «обработка фото» и Midjourney из homepage copy |
| **G4** | Schema parity | FAQ + HowTo в HTML; JSON-LD: `CollectionPage` + `FAQPage` (как на L1-листингах) |
| **G5** | Без каннибализации | Обратный intent «промт из фото» уходит на `/foto-v-promt/` (cross-link + FAQ) |

### 1.3 Жёсткие ограничения

- **Только RU:** тексты в `homepage-seo-copy.ts`. **Не** добавлять i18n/hreflang.
- Орфография в SEO/UI: **«промт/промты»**, не «промпт/промпты» (RU-поиск).
- **Не** менять H2 секций каталога («Люди и отношения», «Стили» и т.д.) — UX-заголовки.
- **Не** дублировать контент `/foto-v-promt/` в hero — только короткий cross-link (уже есть).
- Primary нейросеть в copy: **Nano Banana** (нано банана) + «другие нейросети». **Midjourney не упоминать.**

### 1.4 Разделение intent с `/foto-v-promt/`

| | `/` (главная) | `/foto-v-promt/` |
|--|---------------|------------------|
| Направление | Текстовый промт → генерация фото в ИИ | Фото → текстовый промт |
| Кластер | «промты для фото», «готовые промты», «ИИ-фотосессия» | «фото в промт», «промт из фото» |
| Cross-link | FAQ #6 + ссылка в hero | — |

## 2. Семантическое ядро

### 2.1 Берём в работу

| Запрос | Частота | Где закрываем |
|--------|---------|---------------|
| промты для фото | 78 512 | **H1 + title** (head keyword) |
| промты для фото ии | 11 823 | title, H1 «ИИ-фотосессии», FAQ #4 |
| промты для ии фотосессии | 11 958 | **H1**, description, hero, intro, FAQ #2 |
| промт для создания фото | 7 321 | description, intro, FAQ #3 |
| промт для фото нейросети | 5 678 | H1 «…в нейросетях», description |
| готовые промты для фото | 3 590 | title, hero, FAQ #1 |
| промты для генерации фото | 3 290 | intro, FAQ |

### 2.2 Исключаем

| Запрос / элемент | Частота | Причина |
|------------------|---------|---------|
| промт для обработки фото | 4 382 | Intent = ретушь/Photoshop, не каталог промтов для генерации. Сейчас слово «обработки» ошибочно в description — **убрать** |
| Midjourney | — | Почти нет трафика; достаточно Nano Banana + «другие нейросети» |

### 2.3 Почему «фотосессия» не в title, но в H1

Title зафиксирован под head-кластер (78k + 11k «фото ии»). СЧ «промты для ии фотосессии» (11 958) вынесен в **H1 + description + hero** — title и H1 complement друг друга, а не дублируют одну фразу.

## 3. URL, SEO и метаданные

| Поле | Значение |
|------|----------|
| **Path** | `/` |
| **Canonical** | `https://promptshot.ru/` |
| **robots** | `{ index: true, follow: true }` (без изменений) |
| **Title** | `Промты для фото ИИ — готовые промты для нейросетей \| PromptShot` |
| **Description** | `Готовые промты для фото и промты для ИИ-фотосессии в Nano Banana и других нейросетях. Скопируй промт для создания фото бесплатно — результат за секунды.` |
| **H1** | `Промты для фото` + accent (градиент): `и ИИ-фотосессии в нейросетях` → **«Промты для фото и ИИ-фотосессии в нейросетях»** |
| **Hero subtitle** | `Готовые промты для ИИ-фотосессии и создания фото. Скопируй текст и вставь в Nano Banana или другую нейросеть.` |
| **Open Graph** | `type: website`, `url` = canonical, `siteName: PromptShot`, OG-image с первой карточки (как сейчас) |
| **JSON-LD** | `CollectionPage` (как сейчас) + **`FAQPage`** (новое) |

**Sitemap / robots.ts** — без изменений (главная уже `priority: 1`).

**H1 — почему так:** гибрид закрывает **три кластера** в одной фразе: head «промты для фото» (78k) + СЧ «ИИ-фотосессии» (12k) + «в нейросетях» (5,7k). Орфография: **ИИ-фотосессии** (через дефис). В вёрстке: main `Промты для фото` + accent `и ИИ-фотосессии в нейросетях`.

## 4. Контент страницы

### 4.1 Hero (правки)

- H1: **«Промты для фото и ИИ-фотосессии в нейросетях»** — main `Промты для фото`, accent `и ИИ-фотосессии в нейросетях`
- Subtitle: см. §3
- Chips: «{N}+ **промтов**» (исправить «промптов» → «промтов»)
- Cross-link на `/foto-v-promt/` — **оставить** без изменений

### 4.2 SEO intro (новый блок)

Один абзац под hero (~3–4 предложения), по формуле L1 из `seo-content.ts`:

- библиотека промтов для фото **и ИИ-фотосессий** (портреты, пары, студия)
- проверены в **Nano Banana** (нано банана) и других нейросетях
- бесплатно, на русском, «промт для создания фото»
- **без Midjourney**

### 4.3 HowTo (новый блок)

H2: **«Как использовать промт»**

Переиспользовать шаги из `DEFAULT_HOW_TO` в `landing/src/lib/seo-templates.ts`:

1. Выбери карточку с подходящим промтом и нажми «Скопировать промт».
2. Открой нейросеть (Nano Banana, DALL·E, Flux и др.) или фоторедактор с ИИ.
3. Вставь скопированный текст в поле ввода и добавь своё фото, если нужно.
4. Получи результат и при необходимости скорректируй промт под свой запрос.

*(В шаге 2 допустимо оставить DALL·E/Flux — это не Midjourney; при реализации можно заменить на «другую нейросеть» для единообразия с §1.3.)*

### 4.4 FAQ (6 вопросов)

H2: **«Частые вопросы»**

Разметка `<dl>` как на L1-листингах. Все 6 пар — в `FAQPage` JSON-LD.

| # | Вопрос | Ответ |
|---|--------|-------|
| 1 | Где взять готовые промты для фото? | На PromptShot — библиотека готовых формулировок по категориям: портреты, пары, стили, события. Выберите карточку и нажмите «Скопировать промт». |
| 2 | Где найти промты для ИИ-фотосессии? | В каталоге PromptShot — разделы [«Люди и отношения»](/#audience_tag), [«Стили»](/#style_tag), [«События»](/#occasion_tag) с промтами для портретов, парных и студийных фотосессий в нейросетях. |
| 3 | Как использовать промт для создания фото в нейросети? | Скопируйте промт [с карточки в каталоге](#audience_tag), откройте Nano Banana или другую нейросеть, вставьте текст и при необходимости загрузите своё фото. |
| 4 | Промты для фото ИИ бесплатные? | Да. Все промты на сайте можно копировать и использовать бесплатно. |
| 5 | В каких нейросетях работают промты? | Промты проверены в Nano Banana (нано банана) и подходят для других ИИ-генераторов изображений. |
| 6 | Как получить промт из фото? | Загрузите изображение на странице [Фото в промт](/foto-v-promt/) — сервис вернёт текстовый промт для нейросети. Для разбора картинок на других сайтах — расширение AI Image Describer для Chrome. |

В HTML ответа #6 — `<Link href="/foto-v-promt">`.

#### 4.4.1 Внутренние ссылки в FAQ

**Да — но точечно**, для перелинковки и UX, без переспама.

| FAQ | Что линковать | Куда |
|-----|---------------|------|
| **#2** | Названия разделов «Люди и отношения», «Стили», «События» | Якоря на главной: `/#audience_tag`, `/#style_tag`, `/#occasion_tag` |
| **#3** | «с карточки в каталоге» (одна ссылка) | `/#audience_tag` — первый блок каталога ниже FAQ |
| **#6** | «Фото в промт» | `/foto-v-promt/` (уже в требованиях) |
| **#1, #4, #5** | Без ссылок | Достаточно текста |

**Не линковать:**
- **Nano Banana** и другие нейросети — внешние ссылки уводят с сайта; для SEO главной важнее internal links.
- Каждое слово в ответе — максимум **1–3** internal link на ответ.

**Технически:** на `CategorySection` добавить `id={dimension}` (значения: `audience_tag`, `style_tag`, `occasion_tag`, `object_tag`) — отдельных URL у dimension-заголовков нет, только якоря на главной.

**JSON-LD `FAQPage`:** в `acceptedAnswer.text` — **plain text без HTML** (как на L1); ссылки только в видимом HTML `<dd>`.

### 4.5 Расположение SEO-блоков

**После секций категорий** (`CategorySection`), в конце страницы:

1. SEO intro (`<p>`)
2. HowTo (`<h2>` + `<ol>`)
3. FAQ (`<h2>` + `<dl>`)

Каталог — сразу под hero (поиск → категории).

## 6. Чего не делать

- Не таргетировать «обработка фото» (ни в title, ни в FAQ, ни в description).
- Не упоминать Midjourney в homepage copy.
- Не менять H2 секций каталога.
- Не раздувать hero контентом image-to-prompt (только cross-link + FAQ #6).
- Не добавлять hreflang / `/ru/` prefix.
